

To start this Labwork, you must

1. Open a terminal in the current directory where you found this README file

2. Install the required packages using pip3 and the requirements.txt file:
   For instance on Ubuntu systems, type

     sudo pip3 install -r requirements.txt

3. Run the jupyter notebook

     jupyter notebook

  This should open a new tab in your default browser

4. In your browser tab, click on the jupyter notebook SMS_spams.ipynb
